  • 列表（List）是一种有序和可更改的集合。允许重复的成员。(可变序列)
  • 元组（Tuple）是一种有序且不可更改的集合。允许重复的成员。(不可变序列)
  • 集合（Set）是一个无序和无索引的集合。没有重复的成员。(可变序列)
  • 词典（Dictionary）是一个无序，可变和有索引的集合。没有重复的成员。(可变序列)

方法/其他 操作描述
remove() 一次删除一个元素
重复元素只删除第一个
元素不存在抛出ValueError
pop() 删除一个指定索引位置上的元素

5.列表元素的修改操作

6.列表元素的排序操作

  • 调用sort()方法，列表中的所有元素默认按照从大到小的顺序进行排序，可以指定 reverse = True ,进行降序排序（原列表改变，不会产生新的列表）

  • 调用内置函数 sorted(),可以指定 reverse = True ,进行降序排序，原列表不发生改变（会产生新的列表）

7.列表生成式(生成列表的式子)

[i*i for i in range(1,10)]

  • i*i表示列表元素的表达式
  • i自定义的一个变量
  • range(1,10)是可迭代的对象

9.字典的特点

  1. 字典中所有的元素都是一个key-value对，key不允许重复，value可以重复

2.元组为什么设计成不可变序列

在多任务环境下，同时操作对象时不需要枷锁，因此，在程序中间尽量使用不可变序列。

注意事项：

元组中存储的是对象的引用。

如果元组中对象本身是不可变对象，则不能再引用其他对象。

如果元组中的对象是可变对象，则可变对象的引用不允许改变，但数据可以改变。

3.元组的遍历
