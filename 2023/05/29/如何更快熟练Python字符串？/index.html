二、字符串驻留机制

在Python中字符串是基本数据类型，是一个不可变的字符序列，即不能被修改。

python有一个字符串驻留池，python的驻留机制对相同的字符串只保留一份拷贝，后续创建相同字符串时，不会开辟新的空间，而是把该字符串的地址赋给新创建的变量，如果修改了一个字符串变量的值，那么只是新创建了一个字符串，然后将该变量指向新字符串的地址。

如下示例：

当需要值相同的字符串时，可以直接从字符串池里拿来使用，避免频繁的创建和销毁，提升效率和节约内存，因此拼接字符串和修改字符串是会比较影响性能的。

在需要进行字符串拼接时建议使用str类型的join()方法，而非+，因为join()方法是先计算出所有字符串的长度，然后再拷贝，只new一次对象，效率要比+高。

五、字符串函数

5.1 字符串查询操作

字符串查询操作

find(substr,begin,end)

从左侧查找字符串从指定开始位置到指定结束位置间第一次出现的索引位置，如果查找的子串不存在，则抛出ValueError错误。

rfind(substr,begin,end)

从右侧查找字符串从指定开始位置到指定结束位置间第一次出现的索引位置，如果查找的子串不存在，则返回-1

index(substr,begin,end)

从左侧查找字符串从指定开始位置到指定结束位置间第一次出现的索引位置

rindex(substr,begin,end)

从右侧查找字符串从指定开始位置到指定结束位置间第一次出现的索引位置

count(substr)

查询指定字符串在原始字符串中出现的次数

5.2 字符串状态获取操作

符合条件返回结果为True，否则返回False 。

字符串状态获取操作

islower()

判断字符串是否全部是小写字母组成

isupper()

判断字符串是否全部由大写字母组成

isdigit()

判断字符串是否是由纯数字组成

isalpha()

判断字符串是否由纯字母组成

isalnum()

判断字符串是否由纯数字和字母组成

istitle()

判断字符串是否是单词首字母大小写这种格式

startswith(str)

判断字符串是否以str开始

endswith(str)

判断字符串是否以str结束

isspace()

判断字符串是否全是空白字符。

判断字符串是否以str结束

5.3 字符串转换操作

字符串转换操作

lower()

将字符串中所有字母转成小写字母。

casefold()

将字符串中所有字母转成小写字母,支持各种语言。

casefold()函数与lower()函数类似，将字符串中的所有大写字母转换为小写字母。但是，它还会将一些特定于语言的字符转换为小写形式，例如德语中的ß。因此，casefold()函数比lower()函数更加全面。

upper()

将字符串中所有字母转成大写字母。

swapcase()

字符串中大写字母转成小写，小写字母转成大写。

title()

字符串中每个单词的首字母大写，其余字母小写（区分单词以空格区分）。

capitalize()

字符串首字母大写，其余字母小写。

5.4 字符串格式转换操作

字符串格式转换操作

strip(str)

去掉字符串左右两侧中的str，不填默认为空格。

lstrip(str)

与strip()类似，只是这里只能去掉字符串左侧的字符。

rstrip(str)

与strip()类似，只是这里只能去掉字符串右侧的字符。

ljust(len,str)

在字符串左侧添加len个str字符。

rjust(len,str)

在字符串右侧添加len个str字符。

center(len,str)

在字符串两边添加len个str字符。

zfill(len)

使用0在原始字符串左侧补充到长度为指定值，小数点占1位。

len:补充字符后的字符串长度。

5.5 字符串拆分操作

字符串拆分操作

partition(str)

从字符串左侧查找到参数后，将参数左侧、参数、参数右侧的三个字符串组成元组并返回。

rpartition(str)

从字符串右侧查找到参数后，将参数左侧、参数、参数右侧的三个字符串组成元组并返回。

split(str)

使用参数作为分割线将原始字符串拆分成若干个字符串并组织成列表返回。

不填str参数，默认为空格。

splitlines()

使用**换行分隔符(\n|\r)**作为分割线将原始字符串拆分成若干个字符串并组织成列表返回

字符串连接操作

join(str)

将原始字符串填充到参数的每个字符之间组成新的字符串返回。

5.7 字符串替换操作

字符串替换操作

replace(old_str,new_str,num)

替换原字符串的old_str子串为new_str，num为替换的数量。

expandtabs()

使用空格替换原始字符串中的制表位\t。
